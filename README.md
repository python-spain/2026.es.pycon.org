# 2026.es.pycon.org

Website for the most important Python [event](https://2026.es.pycon.org/) in Spain in the year 2026.

## Requirements

- Node.js v22.0.0 (see `.nvmrc`)
- pnpm
- [pre-commit](https://pre-commit.com/)

## Installation

```bash
pnpm install
pre-commit install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Preview

```bash
pnpm preview
```
